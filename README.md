# WallpaperApp
---

## tips

这是学习Kotlin 和适应java8 Lambda表达式的一个工程。外加mvp+rxjava2+retrofit2+glide。
其中壁纸api主要来自 爱壁纸 应用，通过fiddler 抓取。
